# Sentinels Documentation

**Trust infrastructure for autonomous machines.** This portal documents the Sentinels reference stack.

Sentinels treats every robot or autonomous device the same way a modern web platform treats a user: with a strong identity, signed events, and a verifiable audit trail. The stack is split across small, focused repositories.

## Stack

- [`sentinel-core`](https://github.com/Sentinels-Today/sentinel-core) — Rust trust engine (identity, attestation, trust scoring, audit)
- [`sentinel-cloud`](https://github.com/Sentinels-Today/sentinel-cloud) — Fastify reference API server
- [`sentinel-dashboard`](https://github.com/Sentinels-Today/sentinel-dashboard) — Next.js fleet UI
- [`sentinel-agent`](https://github.com/Sentinels-Today/sentinel-agent) — On-device Rust daemon
- [`sentinel-sdk`](https://github.com/Sentinels-Today/sentinel-sdk) — TypeScript SDK
- [`sentinel-cli`](https://github.com/Sentinels-Today/sentinel-cli) — Operator CLI
- [`sentinel-chain`](https://github.com/Sentinels-Today/sentinel-chain) — Solana attestation anchoring program
